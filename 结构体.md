#### 判断结构体是否为空  
1.基础方法  
```go
if test == (testUser{}) { // 括号不能少
		fmt.Println("test[2] is nil")
	}
  ```  
  
2.使用反射  
```go
type testUser struct {
	id int
	name string
}
func (tu testUser) IsEmpty() bool{
	return reflect.DeepEqual(tu, testUser{})
}
```
